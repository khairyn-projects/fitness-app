# Functional Requirements:
- The system will enable users to register a new account using email and password.
- The system will enable users to log in and out securely.
- The credentials of users can be registered and managed by system admins.
- The system will enable users to record daily workouts.
- The system shall record GPS-based activities (distance, time, pace, route).
- The system shall allow manual activity entry.
- The system will enable users to create and update fitness objectives.
- The system will display reports on workout progress.
- The system will provide reminders to the users.
- Fitness information of users will be stored in the system.
- The system shall allow users to interact with the AI assistant via text input.
- The AI assistant within the system shall generate recommended training plans.

# Non Functional Requirements:
- The  system must be friendly and simple to use.
- The system must be responsive in a few seconds upon interacting with a user.
- The system should provide the security and privacy of the user data.
- The system ought to be mobile friendly.
- The system should maintain stable performance during normal usage.
