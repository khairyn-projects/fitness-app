# User Stories
- As a university student, I would like to track my physical activity.
- As a beginner user, I would like to have a set of fitness goals every week in order to be motivated.
- As a user, I want to record my outdoor physical activities (runs/hikes) using GPS so that I can see my distance and time. When conducting indoor workouts (treadmill, gym lifting) I want to log my statistics manually. 
- As a user, I would like to have reminders of workouts so that I do not forget to work out.
- As a user, I want to view my activity history and performance statistics so that I can monitor my improvement over weeks and months and evaluate my consistency.
- As a user, I would wish to update my personal profile details as this will enable the app to give me the desired tracking results.
- As a user, I want the AI assistant to suggest training plans based on my goals so that I can improve efficiently.
