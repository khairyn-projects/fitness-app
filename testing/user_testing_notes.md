# User Testing Notes

## Overview
User testing was conducted to evaluate the usability and functionality of the Fitness App prototype. Three testers were asked to interact with the prototype and provide feedback regarding the interface, navigation, and overall user experience.

---

## Tester 1
*Name:* Alex  
*Role:* Student

### Tasks Performed
- Register a new account
- Log into the system
- Record a workout activity
- View fitness progress

### Feedback
- The system is easy to navigate.
- The dashboard layout is clear and understandable.
- Recording activities is simple.

### Suggestions
- Add more color or graphics to make the interface more attractive.

---

## Tester 2
*Name:* Sarah  
*Role:* University Student

### Tasks Performed
- Login to the system
- Update fitness goals
- View activity history

### Feedback
- The goal setting feature is useful.
- The activity history is easy to understand.

### Suggestions
- It would be helpful to have more visual charts for progress tracking.

---

## Tester 3
*Name:* Daniel  
*Role:* Gym User

### Tasks Performed
- Log GPS activity
- View progress reports
- Check reminders

### Feedback
- The reminder feature is helpful for maintaining exercise routines.
- The system layout is simple and easy to use.

### Suggestions
- Adding notifications for achievements or milestones could motivate users.
